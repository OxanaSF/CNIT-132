Good Links

         Once we began learning programming, it is hard to stop. The nature of programming captures you and keep you hungry for knowledge with which you can do more exciting things. Additional methods of learning could be using useful coding websites. There are three web sites that have helped me in my learning process, and I believe could be helpful to a new web developer for some point.

  • w3schools

Online tutorial that teaches languages and technologies related to web development. It provides information in concise form, so if someone needs the get quick introduction or refreshment of a particular topic with an example, this is a good sourse.

  • stackoverflow

Probably one of the most popular web site. Primarily known for its section where developers ask a particular questions about a code and get replies with written code and explanation from other developers. Often, after searching your question in google, it sends you to Stackoverflow where you can find exact same or very similar qaustion that had been already asked and answred by someone else.

  • coderbyte

This web site has a great collection of code challenges. I like it for its interface and the challenge structure which incuraged me to solve many of them. As a new developer I was intimidated by famouse leetcode and found Codebyter more friendly for beginners.
